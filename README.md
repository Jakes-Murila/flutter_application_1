# AfyaHive

## AI-Powered Healthcare, Wellness and Community Platform

**Empowering healthier lives through artificial intelligence, preventive healthcare, fitness, and connected communities.**

---

# Overview

AfyaHive is an AI-powered digital health platform built with Flutter that unifies healthcare, wellness, fitness, and community into a single intelligent ecosystem.

Designed with a preventive approach to healthcare, AfyaHive goes beyond treating illness by helping individuals proactively manage their physical and mental well-being. The platform enables users to access healthcare services, monitor their health, improve their fitness, and connect with supportive communities through secure, intelligent, and user-centric technology.

Our long-term vision is to establish AfyaHive as Africa's leading digital health ecosystem and expand globally by delivering accessible, personalized, and technology-driven healthcare solutions.

---

# Installation

Clone the repository:

```bash
git clone https://github.com/Jakes-Murila/afyahive.git
```

Navigate to the project directory:

```bash
cd afyahive
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```

---

# Requirements

* Flutter 3.35 or later
